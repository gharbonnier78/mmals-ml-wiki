# Evidence status taxonomy

| Status | Meaning |
|---|---|
| Foundational | Stable mathematics or canonical project semantics |
| Implemented | Present in a notebook, package, or code path |
| Smoke tested | Executed only as an integrity or feasibility check |
| Multi-seed evidence | Repeated under a declared protocol across seeds |
| Mechanistic support | Diagnostic or causal evidence supports the proposed mechanism |
| Negative result | A declared hypothesis or policy did not obtain the expected result |
| Open hypothesis | A testable design direction without sufficient evidence |
| Speculative branch | Preserved for exploration and clearly outside the validated core |

A page can carry more than one status over time, but the current public badge should state the strongest defensible claim, not the most ambitious aspiration.
