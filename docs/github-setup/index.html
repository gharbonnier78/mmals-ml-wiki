GitHub setup

Publish the repository with GitHub Pages and Actions.

Repository

mmals-ml-wiki
  1. Create the empty repository on GitHub.
  2. Copy the package contents into the repository root.
  3. Commit and push to main.
  4. Open Settings → Pages and select GitHub Actions.
  5. The deployment workflow validates internal links before publication.
