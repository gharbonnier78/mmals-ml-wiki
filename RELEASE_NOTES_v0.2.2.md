# Diderot MMALS/ML Wiki v0.2.2

Minimal registry update for the public `mmals-ecospec` repository.

## Changes

- Adds `mmals-ecospec` to the repository registry as a public EcoSpec 1.0 specification and smoke/reference implementation package.
- Adds a new `ecospec` research branch mapped to TPUT, goal-adaptive control, RC2O, functional memory, geometry, and CAL evidence-contract governance.
- Adds `ecospec-v10` to the publication registry with GitHub-public status and HAL/arXiv/DOI still not submitted/assigned.
- Adds a compact concept page and stable `/go/ecospec` article link.
- Updates branch/repository/publication pages, homepage counts, manifest, and link registry.

## Scientific status

EcoSpec is recorded as a public specification plus executable smoke/reference implementation. It is not yet recorded as comparative evidence that the full architecture outperforms established continual-learning or reinforcement-learning baselines.
