EcoSpec

A minimal MMALS system contract for goal-conditioned, memory-preserving, auditable adaptive learning.

Public specification v1.0reviewed 2026-06-19stable /go/ link

In plain language

EcoSpec turns the MMALS ecosystem into a rulebook: what the system knows, what goal it is pursuing, what route it may choose, when it should abstain, and what trace it must leave behind.

Engineering view

EcoSpec packages typed state, context belief, adaptive complexity mode, goal-conditioned energy routing, dual memory, audit traces, RC2O adapters, RL/forward–backward interfaces, a toy demo, tests, and a trace schema. Its purpose is to make TPUT executable without rewriting every upstream branch.

Research view

EcoSpec is currently a public specification and smoke/reference implementation. It should not be treated as comparative evidence until traces, adapters, baselines, and evaluation contracts have been run on representative settings.
