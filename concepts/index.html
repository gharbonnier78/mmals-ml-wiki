Canonical vocabulary

Concepts

One stable meaning, three reading levels, explicit evidence status, and links to neighboring ideas.

Implemented research program

MMALS

A continual-learning research program in which heterogeneous hosts exchange transformed latent signals through an adaptive, auditable medium.

→Canonical semantics

Biological mapping

A disciplined correspondence between host organisms, the fungal medium, and the mycorrhizal exchange relationship.

→Implemented primitive

Host

A specialist computational module whose value is defined by its contribution to the wider MMALS system, not only by isolated accuracy.

→Implemented primitive

Fungal medium

The active mediator that transforms, routes, allocates, remembers, and audits exchanges among hosts.

→Modeling hypothesis

Mycorrhizal exchange

The governed mutualistic relationship through which hosts and the fungal medium exchange useful resources or information.

→Multi-seed evidence

Inferred context

A distribution q(c|x) over possible situations inferred from the input instead of supplied as a task identifier.

→Implemented primitive

Route

The context- and input-dependent allocation of host contributions used to construct a mediated computation.

→Mechanistic support

Functional route

The composed route-plus-transformation computation, not merely the vector of routing coefficients.

→Multi-seed evidence

Functional memory

Memory that preserves a reusable computation or behavior rather than only parameters, topology, or route identity.

→Design program

Reconstructive and synthetic memory

Two complementary modes: replayable route-function history for audit, and compressed reusable abstractions for efficient operation.

→Partial mechanistic support

Host specialization

Stable, causally meaningful differentiation of host roles across tasks, contexts, or functions.

→Foundational mathematics

Probability simplex

The constrained geometric space in which non-negative route weights sum to one.

→Foundational geometry

Torus

The product of two independent periodic coordinates, S¹×S¹, and a candidate latent geometry only when the data truly carries two periodic degrees of freedom.

→Central evaluated trade-off

Stability–plasticity

The tension between preserving useful old functions and adapting enough to learn new ones.

→Specification and evidence layer

CAL evidence layer

A five-axis evaluation layer separating Competence, Coverage, Compactness, Context, and ACTION.

→System design program

TPUT

The MMALS system-level layer for turning goals, evidence, cost, memory, and routing into governed decisions.
