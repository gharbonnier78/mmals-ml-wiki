CAL evidence layer

A five-axis evaluation layer separating Competence, Coverage, Compactness, Context, and ACTION.

Specification and evidence layerreviewed 2026-06-18stable /go/ link

In plain language

A model can be calibrated without being capable. CAL asks not only whether confidence is statistically valid, but whether the system knows enough to act usefully and safely.

Engineering view

Competence measures task ability; Coverage checks statistical guarantee; Compactness measures set size or decisiveness; Context measures situation inference; ACTION records whether policy gates permit execution. Calibration can restore coverage but cannot manufacture competence.

Research view

CAL prevents conformal or calibration guarantees from being over-interpreted. The axes should be reported jointly, with stale-evidence diagnostics and explicit action policies.

Formal anchor

CAL=(Competence,Coverage,Compactness,Context,ACTION)
