Current external-publication status: the mapped MMALS papers and research packages do not yet have arXiv, HAL, or DOI identifiers in this registry. When a submission occurs, the identifier is added without changing the historical GitHub release date.

Publication mapping contract

  1. A research event points to one or more branches.
  2. A branch points to evidence repositories and publication records.
  3. A publication record keeps separate fields for GitHub, arXiv, HAL, DOI, and venue status.
  4. Scheduled link checks test reachability; scientific status changes only through reviewed metadata.
