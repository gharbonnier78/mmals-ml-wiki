Stability–plasticity

Tune memory pressure and context shift severity to reveal the non-monotonic trade-off.

concept: stability-plasticityteaching simulation
Memory pressure is not a monotonic performance knob. Too little forgets; too much blocks adaptation.

Read the curves

Retention tends to rise with stronger memory, while new-task learning eventually falls. The best operating point depends on the cost of forgetting, the severity of context shift, and the action policy.
