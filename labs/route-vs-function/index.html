Route is not function

Vary route drift and transformation drift independently to see why memory must preserve the mediated computation.

concept: functional-routeteaching simulation
Central distinction: the route can remain unchanged while the transformed host signal changes. Then route drift is zero but functional drift is positive.
route drift Dᵣ
latent drift D𝓏
output drift Dᵧ
diagnosis

Experiment with the counterexample

Set route shift to zero and increase transform drift. The route vector is identical, yet the latent and output computations change. This is why route-vector memory cannot be treated as the complete memory object.
